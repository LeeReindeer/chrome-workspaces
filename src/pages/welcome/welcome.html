Thanks for installing Workspaces!

Take a look how to start using it.

Pin the Workspaces button

To easily access the list of workspaces the Workspaces extension icon should be always visible. Open the Extensions menu, navigate to the Workspaces item and click the Pin icon next to it.

If you prefer keyboard, you can always open the Workspaces menu using Alt + W (on Windows) or ‚å• + W (on Mac).

Workspace for everything

Create a workspace for school subject, personal project, gift idea, vacation plan, or others. All tabs in the workspace are automatically saved. When you open the workspace again tabs are restored to the state where you had left off.

Please note that Tab Groups are not supported by the extension since they are used to mark individual Workspaces.
